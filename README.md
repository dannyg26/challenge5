# MyPiggyBank CTF Challenge 5 (This repository is not part of the problem. The site is simply hosted from here, there is no other relevant information to the challenge in my repository)

This folder contains a **static, fictional demo website** for "MyPiggyBank".

## Safety note
- This template is **not a real bank** and the authentication UI is a **non-functional mock**.
- The pages include a visible "Demo only" notice to discourage entering real credentials.

## Customize photos & bios
1. Replace the placeholder images in `mypiggybank/assets/img/` (keep the same filenames), or update the `<img src="...">` paths in `team.html`.
2. Edit names, roles, and bios directly in `team.html`.

## Files
- `index.html` - Landing page
- `about.html` - About Us
- `mission.html` - Our Mission
- `team.html` - Meet the Team (includes Interns section)
- `contact.html` - Contact Us
- `auth.html` - Sign in / Create account (preview only)
- `assets/css/styles.css` - Styles
- `assets/js/main.js` - Mobile nav + demo modal
